## 修正3: 計算量削減のための最適化

### 修正日
提出形式対応後

### 修正内容

#### 3.1 部位の出現頻度分析機能の追加

**追加したメソッド:**
```python
def analyze_body_parts_frequency(self, sample_size=100):
    """
    データセット全体を分析して、部位の出現頻度を計算
    """
```

**機能:**
- サンプルデータで部位の出現頻度を計算
- 上位N個の部位を選択、または閾値以上の部位を選択
- 計算量削減のため、サンプルデータを使用可能

**修正前の問題点:**
```python
# すべてのカラムを使用
feature_cols = [col for col in df.columns 
               if col not in ['frame', 'mouse_id']]
```

**修正後の改善点:**
```python
# 選択された部位のみを使用
if self.selected_body_parts:
    available_cols = [col for col in self.selected_body_parts 
                     if col in df.columns]
    feature_cols = available_cols
```

**効果:**
- 特徴量数を削減（例: 全部位 → 上位20部位）
- 計算量とメモリ使用量を削減
- ノイズの多い低頻度部位を除外

#### 3.2 アノテーションの出現頻度分析機能の追加

**追加したメソッド:**
```python
def analyze_annotation_frequency(self):
    """
    アノテーションの出現頻度を分析
    """
```

**機能:**
- アノテーションの出現頻度を計算
- 最小出現頻度（デフォルト1%）以上のアノテーションのみを選択
- 低頻度のアノテーションを除外

**修正前の問題点:**
```python
# すべてのアノテーションを使用
for _, row in df.iterrows():
    behavior = row['behavior']
    labels[start:end+1] = self.label_encoder.transform([behavior])[0]
```

**修正後の改善点:**
```python
# 有効なアノテーションのみをフィルタリング
if self.valid_annotations and 'behavior' in df.columns:
    df = df[df['behavior'].isin(self.valid_annotations.keys())]
```

**効果:**
- 低頻度のアノテーションを除外
- データの不均衡を軽減
- モデルの学習を安定化

#### 3.3 データローダーの初期化パラメータ追加

**修正前:**
```python
def __init__(self, data_dir='./data'):
    self.data_dir = Path(data_dir)
    # ...
```

**修正後:**
```python
def __init__(self, data_dir='./data', 
             top_body_parts=None,  # 使用する部位数（上位N個）
             min_annotation_freq=0.01):  # アノテーションの最小出現頻度
    self.data_dir = Path(data_dir)
    self.top_body_parts = top_body_parts
    self.min_annotation_freq = min_annotation_freq
    self.selected_body_parts = None
    self.valid_annotations = None
    # ...
```

**使用例:**
```python
loader = MABeDataLoader(
    data_dir=DATA_DIR,
    top_body_parts=20,  # 上位20個の部位のみを使用
    min_annotation_freq=0.01  # 1%以上の出現頻度のアノテーションのみ
)
```

#### 3.4 load_datasetメソッドの拡張

**修正前:**
```python
def load_dataset(self, video_ids=None, sequence_length=SEQUENCE_LENGTH):
    # 直接データを読み込み
```

**修正後:**
```python
def load_dataset(self, video_ids=None, sequence_length=SEQUENCE_LENGTH, 
                 analyze_frequency=True, sample_size=100):
    # 部位とアノテーションの頻度分析
    if analyze_frequency:
        self.analyze_body_parts_frequency(sample_size=sample_size)
        self.analyze_annotation_frequency()
    # その後、データを読み込み
```

**効果:**
- 頻度分析をオプション化
- サンプルサイズを調整可能
- 計算量を制御可能

### 修正内容

#### 4.1 TensorFlow/Protobuf警告の抑制

**追加したコード:**
```python
import os
import warnings
import sys
from contextlib import redirect_stderr
from io import StringIO

# TensorFlowとprotobufの互換性問題の警告を抑制
os.environ['TF_CPP_MIN_LOG_LEVEL'] = '2'
os.environ['TF_ENABLE_ONEDNN_OPTS'] = '0'
warnings.filterwarnings('ignore', category=UserWarning)
warnings.filterwarnings('ignore', category=FutureWarning)

# stderrを一時的にリダイレクトしてprotobufのエラーを抑制
_stderr_buffer = StringIO()
with redirect_stderr(_stderr_buffer):
    import tensorflow as tf
    tf.get_logger().setLevel('ERROR')
```

**効果:**
- 不要な警告メッセージを抑制
- ログ出力をクリーンに
