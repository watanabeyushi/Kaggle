# 角速度特徴量 実装内容

## 実装日
2025年12月13日

## 概要
回転行動（グルーミング、方向転換、立ち上がりなど）の検出精度向上のため、角速度特徴量を追加しました。

## 実装内容

### 1. 新規関数の追加

#### `add_angular_velocity_features`関数
- **場所**: セル9（新しいセルとして追加）
- **機能**: マウスの回転運動（角速度）を検出する特徴量を生成

**実装詳細**:
- 速度ベクトルから角度を計算（`arctan2`を使用）
- 角度の変化率から角速度を計算（rad/s単位）
- 角度差を[-π, π]の範囲に正規化（360度をまたぐ場合に対応）
- 異常値処理（`winsor_series`で上位0.5%をクリップ）
- 異常値制限（`clip(upper=10.0 * np.pi)`）
- FPS対応（`_scale`関数を使用してウィンドウサイズを調整）

**生成される特徴量**:
- `ang_vel_mean_{15,30,60}`: 角速度の平均（3ウィンドウ）
- `ang_vel_std_{15,30,60}`: 角速度の標準偏差（3ウィンドウ）
- `ang_vel_max_{15,30,60}`: 角速度の最大値（3ウィンドウ）
- **合計9個の特徴量を生成**

### 2. `transform_single`関数への呼び出し追加

- **場所**: セル8の`transform_single`関数内
- **追加位置**: `add_curvature_features`の直後

**変更前**:
```python
        # Advanced features (fps-scaled)
        X = add_curvature_features(X, cx, cy, fps)
        X = add_multiscale_features(X, cx, cy, fps)
```

**変更後**:
```python
        # Advanced features (fps-scaled)
        X = add_curvature_features(X, cx, cy, fps)
        X = add_angular_velocity_features(X, cx, cy, fps)  # 追加
        X = add_multiscale_features(X, cx, cy, fps)
```

## 実装の安全性

既存コードパターンに完全に従った安全な実装：

1. **NaN処理**: `fillna(0.0)`でNaNを0に置換
2. **ゼロ除算防止**: 適切な小さい値（`1e-6`など）を追加
3. **異常値処理**: `winsor_series`で外れ値をクリップ
4. **異常値制限**: `clip()`で物理的に不可能な値を制限
5. **FPS対応**: `_scale`関数でフレームレートに応じたウィンドウサイズ調整
6. **最小期間の設定**: `min_periods`でデータ不足時のエラーを防止

## 期待される効果

1. **回転行動の検出精度向上**
   - グルーミング（体を回転させて毛づくろい）
   - 立ち上がり（後ろ足で立ち上がる動作）
   - 方向転換（急な方向転換）

2. **既存特徴量との相補性**
   - 既存の曲率特徴量（`add_curvature_features`）と相補的な情報を提供
   - 角速度はより直接的に回転運動を捉える

3. **モデル性能向上**
   - 9個の新規特徴量による表現力の向上
   - 特に回転を伴う行動の検出精度が向上することが期待される

## 技術的詳細

### 角速度の計算方法

1. **速度ベクトルの計算**
   ```python
   dx = center_x.diff().fillna(0.0)
   dy = center_y.diff().fillna(0.0)
   ```

2. **角度の計算**
   ```python
   angle = np.arctan2(dy, dx)
   ```

3. **角速度の計算**
   ```python
   angle_diff = angle.diff().fillna(0.0)
   # 角度差を[-π, π]の範囲に正規化
   angle_diff = np.arctan2(np.sin(angle_diff), np.cos(angle_diff))
   angular_velocity = np.abs(angle_diff) * float(fps)  # rad/s
   ```

4. **統計量の計算**
   - 3つのウィンドウサイズ（15, 30, 60フレーム）で平均・標準偏差・最大値を計算
   - ウィンドウサイズはFPSに応じて自動調整

### ウィンドウサイズの調整

- 基準: 30fps
- 実際のFPSに応じて`_scale`関数で自動調整
- 例: 30fps基準の30フレーム → 60fpsでは60フレーム

## 注意事項

- セル9は新しいセルとして追加されており、セルの実行順序に注意が必要です
- 既存の`add_curvature_features`関数とは独立した特徴量を生成します
- 単一マウス用の特徴量であり、ペアマウス用（`transform_pair`）には適用されていません
